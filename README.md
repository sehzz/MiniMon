# MiniMon

**MiniMon** is a lightweight Python-based monitoring system designed to keep an eye on our **MacMini** and connected services.
It performs system health checks, track uptime and monitor key services runnng on custom ports.

---

## Features
- 📊 **System Monitoring**: CPU, memory, disk, and process checks for your Mac mini.  
- 🌐 **Service Health Checks**: Monitors servers running on ports.  
- 🚨 **Alerts & Notifications**: Get notified if services go down or resources hit critical thresholds (under development).  
- ⚡ **Lightweight & Extensible**: Easy to customize with Python for new checks or integrations.
